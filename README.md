## 關於我 About Me

我是來自台灣彰化的視障巡迴輔導教師，長期投入視障教育、低視能學生學習支持、點字教材、觸覺圖形、無障礙網頁與 AI 輔助教材製作。

我關注的不是單純把教材「數位化」，而是讓不同視覺需求的學生，都能用適合自己的方式參與學習。

目前我正在把多年教學現場累積的經驗，逐步整理成可以被複製、改作、部署與分享的無障礙學習工具。

### 無障礙互動教材 Accessible Learning Tools

我正在嘗試將傳統紙本教材轉換成可互動、可朗讀、可鍵盤操作、可調整字級與對比的網頁教材。

設計原則：

- 支援鍵盤操作
- 支援螢幕報讀
- 支援高對比與大字體
- 避免只依賴顏色傳達訊息
- 讓學生可以更公平地完成學習任務

---

### AI × 視障教材製作工作流

我正在整理一套以 AI 協作為核心的教材製作流程：

1. 分析學生需求
2. 蒸餾教師經驗與教學 skill
3. 轉換教材格式
4. 設計無障礙互動介面
5. 測試低視能與螢幕報讀體驗
6. 部署成可分享的網頁作品

這套流程的目標不是取代教師，而是讓教師的專業經驗可以被整理、放大與持續改良。
